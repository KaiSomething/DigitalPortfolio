Boogie

A Discord bot that can play music into voice chat. It was made as a replacement for the late Groovy Bot. It uses the Youtube API to search for music, the YoutubeDownload API to download the mp3 files, and the Discord API to handle commands and play the music. It is written using Node.js.

Commands

-ping: will check if bot is working.
-play [song name/youtube url]: play a song
-stop: stop current song
-leave: kick bot from voice chat
-pause: pause song
-unpause: unpause song
